# IPL Data Analysis

## Project Overview
This project analyzes IPL data using Python and Jupyter Notebook.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Season Wise Winners
- Toss Impact Analysis
- Best Venues
- Player of the Match Analysis
- Top Run Scorers
- Top Wicket Takers
- Team Win Analysis
- Runs Per Over Analysis

## Dataset
- matches.csv
- deliveries.csv
